You can run the demo by launching the SampleApp-EP2.0.exe in
the bin -> release folder.

To compile it you need:

1) The Ogre SDK 1.4.9 VC++ 2005 Express 
You can download it here: http://www.ogre3d.org

2) ETM 2.31
You can download it here: http://www.ogre3d.org/phpBB2addons/viewforum.php?f=16

Enjoy ! :)
